# FruitOps DevOps 

Mini proyecto practico para aprender a usar DevOps usando una api de frutas jajaja

## La chamba

Aprender un poquito como va DevOps construyendo una aplicacion real

- backend con Node.js y Express
- Consumo de una API externa para frutas
- Manejo de errores
- Health check 
- Git y Github
- CI/CD con Github Actions
- Frontend con React
- Deploy
- Docker
